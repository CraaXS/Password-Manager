## Usage

```bash
# Clone this repo
git clone https://github.com/anonymous-inshadows/Password-Manager.git
# Go to the directory
cd Password-Manager
# Compile the code using clang
clang++ -std=c++23 -fuse-ld=lld password-manager.cpp -o password-manager
# Run and use the programm in your terminal
./passord-manager
```

## The options

- You can create a account
- You can save passwords
- You can see the saved passwords
- You can delete passwords

## How it works

- It saves the passwords ENCRYPTED in txt files
- If you want to see the saved passwords you need to enter the account username and password
